# Expense-Tracker

## Description
A console-based application designed to help students track their daily spending. It allows users to add expenses and calculates the total expenditure instantly.

## Features
- Add Expense (Description + Amount)
- View List of Expenses
- Calculate Total automatically

## Tech Stack
- Language: Java
- Tools: VS Code / IntelliJ

## How to Run
1. Download `ExpenseTracker.java`.
2. Open terminal/command prompt.
3. Run: `javac ExpenseTracker.java`
4. Run: `java ExpenseTracker`
